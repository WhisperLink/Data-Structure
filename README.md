# Data-Structure

![C](https://img.shields.io/badge/-C-00599C?style=flat-square&logo=c&logoColor=white)

Basic linked list implementations in C, written for practice.

- **Linked_list.c** - Singly linked list with insert, delete, and print operations.
- **Double_Linked_list.c** - Circular doubly linked list (sentinel head/tail) with add, delete-by-index, print-by-index, and print-all.
